# News Fragments

Add one Towncrier fragment per user-visible change.

This repo uses explicit changelog fragments only. Do not use issue-numbered fragment names.

Use this format:

- `+short-slug.feature.md`
- `+short-slug.bugfix.md`
- `+short-slug.doc.md`
- `+short-slug.removal.md`
- `+short-slug.misc.md`

Examples:

- `+mac-release.feature.md`
- `+pty-runtime.bugfix.md`
- `+release-docs.doc.md`
